# 60-90s Demo Video Shot List

Target audience: developers evaluating multi-agent orchestration frameworks.

## Script

1. Problem framing (0-10s)
   - "Most multi-agent demos look good but are hard to operate in production."
2. Quick start (10-25s)
   - Terminal: `npm install && npm run build && npm run records:api`
3. Trigger one run (25-45s)
   - `POST /api/run/company` with one sentence task.
4. Show observability artifacts (45-70s)
   - Open `session.json`, highlight trace, rounds, model usage.
5. Show final output (70-85s)
   - Open `output.md`.
6. CTA (85-90s)
   - "Star the repo if you want a lightweight but production-minded multi-agent runtime."

## Recording checklist

- Use dark terminal theme and increase font size.
- Keep command history clean.
- Blur any secret keys before publishing.
- Export in 1080p, under 100 MB, with subtitles.
